## PRIORITY 1: Security Fixes (Critical ‚Äî Block Launch)

### 1.1 Hardcoded admin email in client bundle
**Status:** ‚ö†Ô∏è PARTIAL
- `src/lib/admin-auth.ts` correctly uses env var `ADMIN_EMAILS` with server fallback ‚úÖ

**Status:** ‚ùå MISSING
- `src/app/api/auth/register/route.ts` ‚Äî no rate limiting
- No `@vercel/kv` or `upstash/ratelimit` in package.json

### 1.4 No data-access middleman layer
**Status:** ‚ùå MISSING
- No `src/lib/data-access/` directory
- API routes query InsForge directly
**Fix:** Create `src/lib/data-access/products.ts`, `orders.ts`, `users.ts` ‚Äî validate inputs server-side before InsForge queries

### 1.5 bun.lock still in repo
**Status:** ‚ùå
- `bun.lock` exists at repo root

### 1.6 dnd-kit version mismatch
